Abstract

This study develops a data-driven framework integrating online review mining, LDA topic modeling, and a modified quantitative Kano model to identify and prioritize smartwatch user requirements.

Research Gap

  • Limited smartwatch requirement prioritization studies.
  • Traditional methods are often subjective.
  • Limited integration of LDA and Kano evaluation.

Quantitative Kano Model

Kano classification is used to adjust the initial importance score based on customer satisfaction and dissatisfaction effects.

Final Output

  • Requirement priority ranking
  • Function optimization guidance
  • Resource allocation support

Expected Findings

  • Major smartwatch user requirement themes identified.
  • Initial importance calculated using topic frequency.
  • Requirement importance adjusted using Kano satisfaction logic.

Conclusion

The framework supports requirement evaluation, product improvement, and development resource allocation decisions.
